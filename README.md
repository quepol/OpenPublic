# OpenPublic Website

This repository contains the source for my personal website, which describes my advisory work focused on public-sector delivery, leadership, and capacity building.

## What’s in this repository

The site includes:
- A brief overview of my work and focus areas  
- Background on my experience in public-sector digital delivery  
- Descriptions of talks and workshops on leadership, policy, and delivery  
- Contact information  

## How the site is built

- Static site  
- HTML and CSS
- Deployed via GitHub Pages from the `main` branch  

## Status

This site is intentionally simple and will evolve over time. I may add longer-form writing, frameworks, or other public notes here in the future.

How to add a new article

This page is a reference template for site maintenance. To add a new article, open a Claude Code session and say:

"Please use the template to add a new article to the site."

Then provide the following information:

Required

  • Title — Full article title
  • Date — Any readable format (e.g., March 6 2026, 2026-03-06, March 2026)
  • Publication — Where it was published (e.g., Medium, 18F, Ontario Digital)
  • Body — Article content as markdown

Optional

  • Co-author — Co-author name if applicable
  • Source note — Free-text attribution line at the bottom of the article page (e.g., "Originally delivered as a keynote at FWD50 2025.")
  • Original URL — URL used when no explicit source note is provided; generates "Originally published on [Publication]." automatically

Claude Code will derive the slug, formatted date, and card description automatically from the inputs above. The full templates are in the HTML comment at the top of this file's source code.
